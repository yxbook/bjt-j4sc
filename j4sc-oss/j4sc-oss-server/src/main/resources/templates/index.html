公司简介

贵州邦佳大数据科技有限公司2017年成立，坐落于贵州金融城-贵阳国际互联网金融特区，是贵阳市政府招商引资企业；是贵州智慧产业集团核心成员。 公司以智慧城市建设为中心，是一家集智慧园区（社区）服务、互联网平台建设、大数据产品开发、区块链研究为一体的创新型高科技企业。 目前，公司依托强大的研发力量，以“互联网+园区”为切入点，推出了“帮家团”智慧园区综合管理大数据应用平台；同时以门禁、闸机、访客、停车场等系统设备销售安装 为抓手，构筑智能化楼宇物联网体系，为广大物业公司和开发商客户提供了一款优秀的智能化掌上管理应用产品，解决了困扰他们的痛点问题。
